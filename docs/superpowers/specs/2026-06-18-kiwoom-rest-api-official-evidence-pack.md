- Path: `/api/dostk/ordr`
- Method: `POST`
- Production domain: `https://api.kiwoom.com`

## 12. Safety Decision
- v6.1 is evidence-only.
- v6.2 may implement generic broker mock adapter models / guard / fixture using this evidence, but not Kiwoom API execution.
- Kiwoom mock API implementation remains future-only until an explicit opt-in stage.
- Credential handling must remain isolated into a future explicit credential boundary.
- Live order support remains forbidden.
- Parquet remains unsupported.
